# 📈 股票分析与可视化 Skill (Stock Analysis & Visualization)

## 🎯 技能描述

本技能用于分析指定股票的近期走势。它会获取股票的历史交易数据，将**日波动（每日收盘价）**与**周趋势（30日均线）**绘制在同一张图表中进行直观对比。同时，结合当前的股价位置与均线关系，给出客观的买入/卖出时间建议。

## 🛠️ 适用场景

- 用户询问某只特定股票（如 AAPL, TSLA, 600519.SS 等）的最新行情时。
- 用户想知道现在是否是买入或卖出该股票的最佳时机。
- 用户希望看到该股票的可视化走势图。

## 📋 执行步骤

当触发此技能时，请按以下逻辑执行 Python 代码：

1. **获取数据**：根据用户提供的股票代码，拉取最近 3 个月的日线级别历史数据。
2. **计算指标**：
   - 提取每日收盘价，代表“日波动”。
   - 计算 30 日移动平均线（MA30），代表“周波动/中长期趋势”。
3. **绘制图表**：
   - 在同一张图中绘制两条曲线：浅蓝色的每日收盘价折线 + 红色的 MA30 均线。
   - 添加图例、标题和网格，确保图表清晰易读。
4. **生成建议**：
   - **买入信号**：如果最新收盘价 **站上** MA30 均线，说明短期趋势向好，建议“逢低买入”或“持有”。
   - **卖出信号**：如果最新收盘价 **跌破** MA30 均线，说明下行风险较大，建议“卖出”或“暂时观望”。
5. **输出结果**：将生成的可视化图表展示给用户，并附带上述文字版的买卖建议。

## ⚠️ 注意事项

- 如果用户没有提供具体的股票代码，请先礼貌地追问用户想查询哪只股票。
- 所有的投资建议仅供参考，请在回复中提醒用户“股市有风险，投资需谨慎”。
